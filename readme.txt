This plugin allows the editors to search posts/pages with the specific SEO title.

== Installation ==

1. Upload the ss-newsletter-generator folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Adjust the plugin settings
